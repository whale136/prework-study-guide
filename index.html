Prework Study Guide

HTML

  • HTML gives content structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements in header are not visible to user.

CSS

  • CSS gives the content style.

Git

  • git status: check what branch we are currently in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git push origin branch-name: push changes from our local repo.
  • git add -A: stages our modifications/commits our changes

JavaScirpt

  • JavaScirpt handles the logic and allows users to interact with webpage.
